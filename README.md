# Animal Disease Surveillance System

## Overview
The Animal Disease Surveillance System is a web-based platform designed to help monitor and detect animal disease outbreaks. The system allows farmers, pet owners, and citizens to report animal symptoms and helps authorities track disease patterns in real time.

The goal of the project is to improve early detection of disease outbreaks and support faster decision-making for vaccination and preventive measures.

---

## Problem Statement
Animal diseases can spread quickly and affect livestock, pets, and even human health. Traditional reporting systems are slow and lack real-time monitoring, making it difficult for authorities to respond quickly to outbreaks.

---

## Solution
This system provides a digital platform where users can report animal symptoms, and the system analyzes the data to detect possible disease patterns. Authorities can monitor reports through dashboards and take preventive actions.

---

## Key Features
- Community-Based Disease Reporting  
- AI-Powered Disease Detection  
- Interactive Risk Heatmaps  
- Real-Time Alerts & Notifications  
- Centralized Monitoring System for Authorities  

---

## Technologies Used
- React.js
- Node.js
- MongoDB
- TensorFlow
- Twilio API

---

## System Architecture
The system consists of a frontend user interface, backend server, database, and AI model for disease prediction. Data submitted by users is stored in the database and analyzed to detect potential disease outbreaks.
